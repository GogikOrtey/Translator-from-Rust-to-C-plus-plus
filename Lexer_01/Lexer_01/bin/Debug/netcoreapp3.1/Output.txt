Основные настройки: 

> Оборачиваем ли мы на выходе наш код в функцию main : Yes
> Транслятор удаляет неиспользуемые переменные? : No


Входной файл:
fn main() {
    println!("Hello, world!");
}

// Простой пример, который показывает что парсер может обрабатывать простейшие конструкции типа функций

Все распознанные лексемы: 

0    fn         -  FUNCTION
1    main       -  MAIN
2    (          -  OPEN_BRACKET
3    )          -  CLOSED_BRACKET
4    {          -  START_VOID
5    println!   -  PRINT
6    (          -  OPEN_BRACKET
7    "Hello, world!"  -  STRING
8    )          -  CLOSED_BRACKET
9    ;          -  END_LINE
10   }          -  END_VOID
 
Дерево разбора парсера: 
 
FUNCTION
  NAME : MAIN
  ARGUMENT
    NO_ARGUMENTS
  BODY
    PRINT
      ARGUMENTS
        STRING : "Hello, world!"
END_FUNC
